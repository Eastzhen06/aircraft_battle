# 飞机大战 (Plane War) - 项目功能说明文档

**版本:** 1.0.0  
**日期:** 2026-02-15  
**生成时间:** 2026-02-15

---

## 1. 功能列表

本项目是一款基于 HTML5 Canvas 的现代版“飞机大战”游戏，包含以下核心功能模块：

*   **多游戏模式系统**：包含经典闯关、无尽挑战及双人合作模式。
*   **战机与技能系统**：5种具有独立属性和特殊技能的战机。
*   **僚机系统**：3种不同功能的僚机（攻击、防御、辅助）。
*   **经济与升级系统**：金币/钻石经济体系，支持战机30级属性强化。
*   **战斗核心机制**：碰撞体积检测、互伤机制、悬浮战斗数值反馈。
*   **开发者调试系统**：内置开发者后门，用于快速测试和解锁内容。
*   **用户界面 (UI)**：包含主菜单、机库、商店、关卡选择及战斗HUD。

## 2. 功能详细描述

### 2.1 游戏模式 (Game Modes)

*   **闯关模式 (Campaign Mode)**
    *   **描述**: 包含30个精心设计的关卡，难度逐级递增。
    *   **机制**: 玩家需击败关卡内的普通敌机及最终Boss方可通关。
    *   **解锁**: 需按顺序通关解锁下一关。

*   **无尽模式 (Endless Mode)**
    *   **描述**: 无限生成的敌机波次，挑战玩家的生存极限。
    *   **机制**: 游戏每进行60秒自动刷新一个Boss，随着时间推移敌人强度不断提升。
    *   **目标**: 尽可能获得高分，刷新历史记录。

*   **双人合作模式 (Co-op Mode)**
    *   **描述**: 本地双人同屏游戏。
    *   **机制**: P1 和 P2 共同迎战敌人，共享屏幕空间，需配合走位与输出。

### 2.2 战机系统 (Aircraft System)

游戏包含5种不同定位的战机，每种战机拥有独立的属性成长和特殊技能（按 `E` 键释放）：

1.  **战斗机 (Fighter)**
    *   **定位**: 均衡型
    *   **技能**: **火力全开 (Firepower Boost)** - 短时间内大幅提升射速和伤害。
2.  **拦截机 (Interceptor)**
    *   **定位**: 高机动型
    *   **技能**: **导弹锁定 (Lock-on)** - 发射自动追踪导弹打击屏幕内敌人。
3.  **轰炸机 (Bomber)**
    *   **定位**: 重装高伤型
    *   **技能**: **地毯式轰炸 (Carpet Bomb)** - 对全屏范围造成毁灭性打击。
4.  **护卫机 (Guardian)**
    *   **定位**: 防御生存型
    *   **技能**: **护盾共享 (Shield Share)** - 生成强力护盾抵挡伤害（双人模式下可保护队友）。
5.  **隐形机 (Stealth)**
    *   **定位**: 战术突袭型
    *   **技能**: **隐形突袭 (Stealth Raid)** - 进入隐身状态并对穿越的敌人造成伤害。

### 2.3 经济与升级系统 (Economy & Progression)

*   **货币体系**:
    *   **金币 (Gold)**: 击杀敌机掉落，用于常规升级。
    *   **钻石 (Diamond)**: 击败Boss或充值（模拟）获取，用于购买高级物品。
*   **30级强化系统**:
    *   玩家可在“机库”中消耗金币提升战机等级。
    *   **属性提升**: 每次升级增加生命值、攻击力、防御力。
    *   **消耗曲线**: 1-5级为固定消耗，6-30级消耗随等级线性增长。

### 2.4 开发者/管理员系统 (Developer Tools)

*   **功能**: 快速解锁游戏所有内容以便测试。
*   **入口**: 点击主界面右上角的头像图标。
*   **账号**: `0001`
*   **密码**: `1011`
*   **效果**: 登录成功后，解锁所有战机、解锁全部30个关卡、获得大量金币与钻石。

### 2.5 战斗体验优化

*   **碰撞互伤**: 玩家与敌机相撞时，敌机受到巨额伤害（通常直接摧毁），玩家受到固定伤害（30点），避免了“穿模”现象。
*   **视觉反馈**: 拾取道具（如护盾、加血）时，战机上方会弹出悬浮文字提示（如 "SHIELD!", "HP +30"）。
*   **大型战机**: 玩家战机尺寸调整为 80x96，视觉效果更显著。

## 3. 技术实现概述

*   **核心引擎**: 原生 HTML5 Canvas API，无第三方游戏引擎依赖。
*   **架构模式**:
    *   **ECS (Entity-Component-System) 变体**: 实体（Player, Enemy）、组件（Stats, Position）、系统（Collision, Input）分离。
    *   **面向对象 (OOP)**: 使用 ES6 Class 封装 `Game`, `Player`, `Enemy` 等核心类。
*   **数据持久化**: 使用 `localStorage` 保存玩家进度、金币及解锁状态。
*   **模块化**: 采用 IIFE (立即调用函数表达式) 和 ES6 模块管理代码，避免全局变量污染。
*   **资源管理**: `ImageLoader` 和 `SoundManager` 负责异步加载图片与音频资源。

## 4. 用户交互说明

### 4.1 键位操作

| 功能 | 玩家 1 (P1) | 玩家 2 (P2) |
| :--- | :--- | :--- |
| **移动** | W, A, S, D | 方向键 (↑, ↓, ←, →) |
| **射击** | 空格键 (Space) | 回车键 (Enter) |
| **技能** | E | (暂无独立技能键) |
| **暂停** | P | - |

### 4.2 界面交互

*   **主菜单**: 点击按钮选择模式或进入子菜单（机库、商店）。
*   **登录**: 点击右上角头像 -> 输入账号密码 -> 点击登录。
*   **机库**: 点击战机图标切换选择，点击“升级”按钮强化战机。
*   **战斗中**:
    *   左上角显示血量条和护盾状态。
    *   右上角显示当前得分。
    *   拾取掉落物（星星、医疗包、闪电）自动生效。
